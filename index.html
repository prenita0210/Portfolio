About Me

Hello! I'm Prenita Darshani, a passionate and detail-oriented Front-End Developer based in Pune.
I have over 2 years of experience building responsive and user-friendly web applications using technologies like Angular, HTML5, CSS3, JavaScript, and Bootstrap.

I started my journey in tech at Amazon as a Process Associate, where I contributed to internal UI tools using front-end technologies. Since then, I’ve worked on various mini projects, including an Admin Login System, Angular Chat App, and a Product Review Page, which helped me sharpen my skills in form validation, API integration, and responsive design.

I’m enthusiastic about clean code, user experience, and staying up-to-date with the latest front-end trends. I'm also exploring project coordination and leadership, aiming to bridge the gap between development and business goals.

When I'm not coding, I enjoy reading about new web technologies, solving coding challenges, and contributing to side projects.
